# Useful libraries, resources, and links for Mojo

::: tip Official resources for Mojo

Here are some official resources for Mojo that you may find useful:

1. [Mojo official manual](https://docs.modular.com/mojo/): Comprehensive information about the language, its features, and change log.
1. [Mojo standard library Github repository](https://github.com/modular/modular): The official repository for the Mojo standard library.
1. [Mojo community forum](https://forum.modular.com/): A place to post questions, share knowledge, and discuss Mojo.
1. [Mojo Discord server](https://discord.gg/m4uBMDNv): The Discord server to discuss Mojo, ask questions, and get help from others.

:::

::: warning Libraries for Mojo

Here are some useful libraries for Mojo. They are written in pure Mojo and are generally ready to use. Some provide functionality similar to modules in Python's standard library that Mojo does not yet have.

**Mathematical and numerical** libraries:

1. [NuMojo](https://github.com/Mojo-Numerics-and-Algorithms-group/NuMojo): N-dimensional array type and numerical algorithms, similar to NumPy.
1. [Decimo](https://github.com/forfudan/decimo): Arbitrary precision integer and decimal library for Mojo, similar to Python's `int` and `decimal.Decimal` types.

**Command-line** libraries:

1. [ArgMojo](https://github.com/forfudan/argmojo): A command-line argument parsing library for Mojo, similar to Python's `argparse`.
1. [Mist](https://github.com/thatstoasty/mist): An terminal toolkit providing text styling, terminal control, event reading, etc.

**File parsing** libraries:

1. [EmberJson](https://github.com/bgreni/EmberJson): A JSON parsing library for Mojo, similar to Python's `json` module.

:::

::: tip Other useful resources and links for Mojo

1. [awesome-mojo](https://github.com/mojicians/awesome-mojo): A curated list of awesome Mojo frameworks, libraries, software and resources.

:::
